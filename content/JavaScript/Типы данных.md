## Поддерживающиеся типы данных
Стандарт ECMAScript определяет 8 типов:
- [[undefined]]
- [[Boolean]]
- [[Number]]
- [[String]]
- [[BigInt]]
- [[Symbol]] #ES6 
- [[Null]]
- [[Object]]

Для определения типа данных используется `typeof`

А для определения типа структур используется `instanceof`
