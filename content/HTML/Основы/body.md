#блочный 

Элемент `<body>` содержит весь контент, который появится на странице: текст, изображения, теги, скрипты JavaScript и так далее.

В вашем документе может быть только один `<body>`

## Стили по умолчанию
```css
body {
    display: block;
    margin: 8px;
}
```
